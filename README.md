# instagram-influencer-sql-project

## 📌 Overview
This project analyzes influencer engagement data using SQL.  
It simulates a social media platform with users, posts, likes, and comments.

---

## 🛠️ Tools Used
- SQL (MySQL)

---

## 📊 Database Design
The project includes the following tables:
- Users
- Posts
- Likes
- Comments

---

## 🔍 Key Analysis Performed
- Calculated total likes per post  
- Identified total engagement per user  
- Found top influencer based on likes  
- Compared performance of different content types  

---

## 📈 Sample Insights
- Reels received higher engagement compared to other content types  
- Top influencer identified based on total likes  
- Engagement varies depending on content type  

---

## 🚀 Skills Demonstrated
- SQL Joins  
- GROUP BY & Aggregations  
- ORDER BY & LIMIT  
- Database design and relationships  

---
